Esta aplicación tiene como objetivo principal proporcionar un formulario con validaciones, proteger la URL de acceso, desplegar capacitaciones y permitir la creación de nuevas capacitaciones desde una base de datos.

Características principales
Validaciones en el formulario: El formulario implementa validaciones para asegurar que los datos ingresados sean correctos y consistentes. Esto ayuda a mantener la integridad de la información y prevenir errores.

Protección de la URL: La aplicación cuenta con medidas de seguridad para proteger la URL de acceso. Esto garantiza que solo los usuarios autorizados puedan acceder y utilizar la aplicación.

Despliegue de capacitaciones: Podrás visualizar una lista de capacitaciones disponibles en la aplicación. Esta funcionalidad te permitirá explorar y acceder a las capacitaciones existentes de manera sencilla.

Creación de capacitaciones: Además de las capacitaciones preexistentes, podrás crear nuevas capacitaciones directamente desde la base de datos. Esto te brinda la flexibilidad de adaptar la aplicación a tus necesidades específicas.

Se adjunta además el script .sql para generar la base de datos utilizada. Es importante notar que para el usuario utilizamos root además de la contraseña 'Grupo7' a modo de no tener que portar dichos datos en la conexión cada vez que quisiéramos probar el último push.
